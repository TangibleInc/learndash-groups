Groups Extended for LearnDash

== Description ==

Groups Extended for LearnDash is a free plugin developed by Tangible Plugins that allows you to display a group page to group members, providing a space to  display group-related content.

== Installation ==

1. Install & activate in the admin: *Plugins -&gt; Add New -&gt; Upload Plugins*
